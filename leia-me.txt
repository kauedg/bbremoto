1. Informação geral
O programa “BB Remoto – VPN – SNX” é uma interface gráfica que permite e facilita o uso do programa SSL Network Extender (SNX) da empresa Check Point Software Technologies Ltd.

2. Instruções sobre configuração
Para que o “BB Remoto – VPN – SNX” possa funcionar corretamente é necessária a prévia instalação do SNX em seu sistema operacional Linux, caso não tenha realizado a instalação, favor seguir o passo a passo descrito no arquivo PDF que consta no arquivo “bbemoto.zip” baixado.

3. Instruções sobre instalação
A instalação será realizada por arquivo de script, para isso é necessário extrair apenas o arquivo “instalar.sh” para a mesma pasta onde está localizado o arquivo “bbremoto.zip”.

Será necessário abrir um terminal para executar dois comandos, o primeiro será para conceder permissão de execução do script de instalação e o segundo será o comando de execução do arquivo de instalação, certifique-se de que o terminal está aguardando comando na pasta onde está o arquivo “instalar.sh”

OBS: Não será necessário utilizar os comandos como superusuário.

Primeiro comando: chmod +x instalar.sh
Segundo comando: ./instalar.sh

Aguarde até ser exibido no terminal a mensagem: “Sucesso em copiar os arquivos para a pasta: /~/.local/share/bbremoto”

4. Instruções sobre como operar o programa
Existem dois campos e eles são obrigatórios, o primeiro é “Senha do certificado”, onde deve ser digitado a senha do certificado que será utilizado para permitir o acesso à VPN e o segundo campo é “Certificado selecionado”, onde será aberta janela para localizar e selecionar o certificado de extensão “.p12” que permite o acesso à VPN.

5. Manifesto de arquivos
bbremoto.glade – Arquivo de desenvolvimento da interface gráfica.
bbremoto.sh – Arquivo script que possui comando para executar o programa “BB Remoro”.
BB Remoto – Arquivo com código fonte (Python) que permite a integração entre a interface gráfica e o programa SNX.
BB Remoto - VPN - SNX.desktop – Arquivo lançados que permite executar o “bbremoto.sh”.
instalar.sh – Arquivo script que executa comando de extração e cópia dos arquivos para a pasta: “/~/.local/share/bbremoto”.
logobb.png – Arquivo de imagem utilizado para identificar o ícone do “BB Remoto” e na tela “Sobre”.
.p12.conf – Arquivo que será gerado após selecionar o certificado, tendo como conteúdo o caminho onde está armazenado o certificado, para que possa ser utilizado nas próximas utilizações do programa.
